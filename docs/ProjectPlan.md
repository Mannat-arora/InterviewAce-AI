# 📋 InterviewAce AI - Project Plan

## 🎯 Problem Statement

Students preparing for placements often use multiple platforms for coding practice, resume analysis, HR interview preparation, and company-specific interview preparation.

InterviewAce AI aims to combine these into one intelligent platform.

---

# 👥 Target Users

- College Students
- Placement Aspirants
- Fresh Graduates

---

# ✨ Core Features

## Phase 1
- User Authentication
- Dashboard
- User Profile

## Phase 2
- DSA Progress Tracker
- Daily Goals
- Coding Statistics

## Phase 3
- Resume Analyzer

## Phase 4
- AI HR Interview

## Phase 5
- Technical Interview Questions

## Phase 6
- Company-wise Preparation
